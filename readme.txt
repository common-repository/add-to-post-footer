Add some content to the bottom of all your posts. You can add images and HTML too. Install and activate it, it's easy!

== Description ==

How to add some content to all posts? Easy, install this plugin to your website/blog. When activated, a new option in the Posts menu will be able, Add To Post Footer option. Click there and insert the content what you want to add to all posts footer. Save it and it's done. As you activate the plugin, all posts will have the content. If you don't want more the content, just deactivate the plugin.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Posts->Add To Post Footer screen to choose the content would be added.
4. Click on Save button.
